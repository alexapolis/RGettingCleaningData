#run_analysis.R

* load dplyr
* Download data
* Unzip
* Load test and training data and bind rows
* Load features data and attach readable variable names to columns
* Load activity label data and translation table, bind test and training sets
* Attach human readable Activity name to data rows as a new column
* Load subject data, bind test and training sets
* Attach Subject ID to data rows as a new colum
* Strip away columns not used for mean or standard deviation for each variable -> ActivityData
* Group data by Subject and Activity
* Summarize data by storing the mean for grouped rows in each variable -> ActivityMeans
